## Scripts
Place yourself in the root directory and run the following commands.
```
$ ./scripts/create.sh
```
or
```
$ ./scripts/train.sh
```
or just ask the script for help
```
$ PYTHONPATH=. python src/main.py -h
```
N.B. remember to give execute rights.
